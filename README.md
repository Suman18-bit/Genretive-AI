## ⚡ Quick Start

</div>

Get up and running in your local environment in under 5 minutes:

### 1️⃣ Clone the Repository
```bash
git clone [https://github.com/Suman18-bit/Genretive-AI.git](https://github.com/Suman18-bit/Genretive-AI.git)
cd Genretive-AI

```

### 2️⃣ Set Up Virtual Environment

```bash
# Create virtual environment
python3 -m venv venv

# Activate 
source venv/bin/activate

```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt

```

> 💡 **Hardware Note:** If you are running locally (e.g., i5 12th Gen configurations), ensure your Python environment is optimized. For GPU-accelerated workloads, check `requirements-gpu.txt`.

### 4️⃣ Launch the Interface

```bash
# To run predictive models or Jupyter Notebooks:
jupyter lab

# OR launch a Streamlit deployment:
streamlit run app.py

```
